## About this theme

Buttercut is a theme-first Next.js starter. All copy you see here — including this page — is demo content you can replace under `content/demo/`.

## What ships by default

- A typed `site.config.ts` with sensible defaults for navigation, socials, and home-page blocks.
- A block registry so you can swap built-in blocks (hero, demo projects, Now Playing, weather, integration summary) without forking.
- Optional integrations that stay off until you flip them on in `site.config.ts` and add environment variables. With everything off, the site still builds and runs.

## Typography and UI

Editorial "magazine" cards, a wave-underline navigation, and a Nunito / Bitter / JetBrains Mono font stack. Light + dark + system theme with no flash on load.
